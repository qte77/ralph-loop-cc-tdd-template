# DevContainer Configurations

VS Code prompts to select configuration on first open.

## Active Configurations

**`template/`** - Template Development
- Edit template files (markdown, YAML, scripts)
- Contribute to Ralph Loop template
- Alpine + vscode user (non-root)
- **Size:** ~10MB

**`project/`** - Project Development
- Build your application from this template
- Python + Node.js + Docker toolchain
- **Size:** ~1GB+

## Quick Decision

Template work? → `template/` | Project work? → `project/`

## Switching

`Cmd/Ctrl+Shift+P` → "Dev Containers: Rebuild and Reopen in Container"
